과제: 배운 내용 복습 정리

1. 웹 개발 기초 개념
- IDE: 서버가 읽을수 있는 언어(코드)를 입력할수있는도구
- 브라우저: 개발자가 코드를 이용하여 서버에 등록하면 실제이용자가 볼수있도록 변경해주는 중간통역사 같은 느낌
- 웹: 인터넷에 등록되있는 모든 페이지망 

2. Git과 GitHub
- Git: 내pc에서 작성 , 수정한 코드를 깃허브로 올리기 위한 도구 깃허브에 올리기 전단계로 되돌리기(복구)기능이 가장핵심!!
- GitHub: 내가 만든 코드파일을 공유할수있도록 만든 클라우드 같은 목표를 가진 팀원들과 서로의 코드를 공유 컴펌받을수있음 
- 스테이징: 겟허브에 올릴 코드들을 미리 담아두는것 
단축키는 git add 파일명 
- PR (Pull Request): 내가 만든 코드를 최종적으로 검토받고 합치기 위해 요청하는 것

3. 주요 Git 명령어
- git init: 시작하기 (설명필요!)
- git status: 상태 확인하기 (설명필요!)
- git add: 장바구니 담기 
- git commit -m "메시지" 겟허브에 같은카테고리 끼리 묶음으로 저장하기 (-m 으로 메세지 전달기능있음)
- git push: 겟허브에 내 코드파일 업로드하기
- git pull: 겟허브에서 코드파일 가져오기
